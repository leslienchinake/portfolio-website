# 🌐 Personal Portfolio Website

## 📌 Project Description
This is a responsive personal portfolio website created as part of a Web Technologies practical assignment. The website showcases my skills, projects, and contact information in a clean and user-friendly design.

---

## 🚀 Features
- Home page with hero section, tagline, and call-to-action button
- About page with biography, skills, and education timeline
- Projects page displaying at least 3 projects using CSS Grid
- Contact page with a functional form and validation
- Fully responsive design (mobile, tablet, desktop)
- Social media links and email integration

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
